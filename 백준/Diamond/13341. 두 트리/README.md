### 문제 설명

<p>정점의 개수가 N개인 두 트리가 주어진다. 트리의 각 노드는 0부터 N-1까지 번호가 매겨져 있다.</p>

<p>i번 정점의 점수는 S<sub>i</sub>이다.</p>

<p>집합 {0, 1, ..., N-1}의 부분 집합을 골라서 부분 집합에 포함되어 있는 정점의 점수의 합의 최댓값을 구하는 프로그램을 작성하시오. 이때, 다음과 같은 두 조건을 만족해야 한다.</p>

<ul>
	<li>첫 번째 트리에서 부분 집합에 포함되어 있는 정점끼리 연결 서브 그래프를 만들어야 한다.</li>
	<li>두 번째 트리에서 부분 집합에 포함되어 있는 정점끼리 연결 서브 그래프를 만들어야 한다.</li>
</ul>

<p>두 조건을 만족시키는 부분 집합 중에서 정점의 점수가 가장 큰 것을 구하는 프로그램을 작성하시오.</p>

### 입력 

 <p>첫째 줄에 정점의 개수 N (2 ≤ N ≤ 50)이 주어진다.</p>

<p>둘째 줄부터 N-1개 줄에는 첫 번째 트리의 간선 정보가 주어진다.</p>

<p>그 다음 N-1개 줄에는 두 번째 트리의 간선 정보가 주어진다.</p>

<p>마지막 줄에는 점수 S<sub>0</sub>, S<sub>1</sub>, ..., S<sub>N-1</sub>이 주어진다. (-1,000 ≤ S<sub>i</sub> ≤ 1,000)</p>

### 출력 

 <p>첫째 줄에 점수의 최댓값을 출력한다.</p>
